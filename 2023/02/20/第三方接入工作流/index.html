1. 启动

POST: /runtime/instance/v1/easywebWorkflowStart

请求头参数：

参数名 说明 值
Authorization 用户认证信息 Bearer 用户认证信息
Content-Type 类型 application/json

最请求参数示例：

{
         "formType":"frame", //表单类型
         "businessKey":"17231238972", //业务主键
         "flowKey":"cs1", //流程key
         "account":"admin", //账号登陆名
         "sysCode":"123456" //额外参数，用于区分不同的流程或者系统
         "vars": {     //设置流程变量
          "additionalProp1": "string",
          "additionalProp2": "string",
          "additionalProp3": "string"
        }
}
